# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="hex-rgba.js"></script>
<hex-rgba></hex-rgba>
```

## React

```jsx
import "@sgbp/hex-rgba";
export default function Page() { return <hex-rgba />; }
```

## Vue

```vue
<script setup>
import "@sgbp/hex-rgba";
</script>

<template>
  <hex-rgba />
</template>
```
