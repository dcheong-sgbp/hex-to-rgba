# Hex to RGB / RGBA Converter

A hex to RGB/RGBA converter: paste a hex colour, set the alpha, and copy the rgb() and rgba() values with a live swatch. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="hex-rgba.js"></script>
<hex-rgba></hex-rgba>
```

## What it does

RGB sets red, green and blue channels (0-255); RGBA adds an alpha channel for opacity (0-1). So rgba(235, 0, 40, 0.5) is the same colour at 50% transparency. 3-digit shorthand like #fff expands to #ffffff.

## When you'd use it

Any time you need a brand colour at partial opacity — overlays, hovers, shadows, frosted-glass panels.
